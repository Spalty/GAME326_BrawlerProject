REQUIRED ASSETS - Create these in Unity
=======================================

The following ScriptableObject assets need to be created in this folder:

1. MatchConfig.asset
   - Right-click > Create > Brawler > Match Config
   - Configure rounds to win, timing, etc.

2. InputConfig.asset (if using)
   - Right-click > Create > Brawler > Input Config
   - Configure input buffer times

See Lesson 02 for MatchConfig setup details.
