EXAMPLE ATTACK ASSETS - Create these in Unity
==============================================

Create the following AttackData ScriptableObjects here:

1. Jab.asset
   - Right-click > Create > Brawler > Attack Data
   - Fast, low damage, low knockback
   - See Lesson 05 for recommended values

2. ForwardAttack.asset
   - Medium speed and damage
   - Good horizontal knockback

3. UpAttack.asset
   - Launcher - high vertical knockback angle

4. DownAttack.asset
   - Spike potential (negative Y knockback when aerial)

5. AerialAttack.asset
   - General purpose air attack

These will be assigned to the ExampleFighter prefab's AttackController.
See Lesson 05 for detailed attack setup instructions.
